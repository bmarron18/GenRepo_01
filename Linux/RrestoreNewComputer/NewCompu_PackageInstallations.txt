===== gdebi ==================

If your current system already has all the prerequisite packages installed, then life is good. 
You simply download the deb file, and install it with the dpkg command:

$ sudo apt update
$ sudo dpkg -i somelocal.deb

But, if you run into package dependency problems, you are on your own. 
The dpkg command does not resolve dependency problems for you.
For a graphical version run gdebi-gtk


Use Software Center

OR

sudo aptitude install gdebi-core
sudo apt update
sudo apt install gdebi
